3. What is the difference between   "this" and  $(this) in jquery programming?
A. this is the DOM object, whereas $(this) is the jQuery wrapper around same.
   this returns dom object
   $this returns jQury object
   When using this, you can call DOM methods on it, but not jQuery methods. 
   When using $(this), you can call jQuery methods on it, but not DOM methods.
